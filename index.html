pyglmnet 0.1.dev0¶

pyglmnet is a Python library implementing elastic-net regularized generalized linear models (GLM). Notations and approach are followed as in Friedman, J., Hastie, T., & Tibshirani, R. (2010). The implementation is aim to suite users the same way along side with popular GLM R package. The only different is that we still use batch gradient descent rather than coordinate gradient descent which os very fast for n_samples x n_features size up to 10000 by 1000.

Introduction¶

In Generalized Linear Model (GLM), we mainly want to solve the following problem.

\[\min_{\beta_0, \beta} \frac{1}{N} \sum_{i = 1}^N w_i \ell (y_i, \beta_0 + \beta^T x_i) + \lambda [0.5(1 - \alpha)\| \beta \|_2^2 + \alpha \| \beta \|_1]\]

where \(\ell (y_i, \beta_0 + \beta^T x_i)\) is negative log-likelihood of an observation \(i\)

Installation¶

You only need numpy, scipy and (optional) scikit-learn as requiments. You can use pip to install all dependencies,
pip install numpy
pip install scipy
pip install scikit-learn

To install pyglmnet, first clone the repository

git clone http://github.com/pavanramkumar/pyglmnet

Then run setup.py to install the library

python setup.py develop install
